# champions_league_data_analysis
Champions League SQL Analysis Project: Built a relational database using UEFA Champions League data (1991–2024), performing SQL queries to analyze team performance, player statistics, and tournament trends.


UEFA Champions League Seasons Analysis (1991–1992 to 2023–2024)

This project focuses on analyzing UEFA Champions League data from the 1991–1992 season to the 2023–2024 season using MySQL. The database consists of multiple related tables containing information about teams, players, seasons, group stages, knockout stages, and squad details. The project demonstrates database design, data import, SQL querying, and analytical reporting techniques.

Key analyses include identifying the most successful teams, tracking club appearances across seasons, analyzing group-stage performance, evaluating knockout-stage progression, and examining player participation through minutes played. Various SQL concepts such as joins, aggregate functions, subqueries, grouping, sorting, and filtering were used to generate meaningful insights from the dataset.

This project helped strengthen skills in database management, data cleaning, relational schema design, and business intelligence reporting. It serves as a practical example of applying SQL to real-world sports analytics and extracting valuable insights from historical football data.
